# aoc2023
Advent of Code 2023

This is my repo for solutions to this year's Advent of Code competition.

I started out writing the solutions in C, so I guess that's what I'll continue to use.  We'll see how far I get before I become annoyed with the problems.

## Update 9 Dec 2023

Well, I finished the solutions through Day 6 and the first problems of Day 7 and Day 8.  But, I don't want to go too much further.

Day 5b is certainly the talk of a small group of developers including me.  I brute-forced my way to the end in 9 minutes, but some of the others are using multiple threads to attack the problem.  I started down that path until I found that the MinGW gcc compiler that I'm using doesn't seem to have POSIX threads support.   I think I will revisit this problem in a number of forms, so trying out the competition was worth it for me.
